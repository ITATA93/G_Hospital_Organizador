## Cómo Documentar un Endpoint

1. Copia la plantilla:
   ```bash
   cp endpoint-template.md endpoints/mi-endpoint.md
   ```

2. Completa todas las secciones siguiendo el formato estándar

3. Actualiza este README agregando el enlace en la categoría apropiada

## Herramientas Recomendadas

- **Postman**: Importar `postman/collection.json`
- **Swagger**: Generar desde schemas
- **Thunder Client**: Extension de VS Code
